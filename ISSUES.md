# RIS — Issues & Feedback Manual

Noteaza aici problemele gasite la utilizare. Format minim — Claude citeste si diagnosticheaza.

## Cum scrii

```
## [Data]
- [Ce ai facut]: [Ce s-a intamplat]
```

Exemple:
```
## 2026-03-24
- Analiza CUI 4239234: HTML OK, dar PDF nu are tabele
- Pagina Compare: buton "Genereaza" nu face nimic
- Dashboard: graficul trend arata 0 pe toate lunile
```

Nu trebuie sa explici tehnic. Scrie CE ai facut si CE ai vazut. Claude cauta in log-uri cauza.
